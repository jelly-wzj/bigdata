java虚拟机是如何加载ＣＬＡＳＳ文件，ＣＬＡＳＳ文件信息进入虚拟机后会发生什么样的变化？



##### 类的加载方式

* 静态加载

  ```java
  Person p = new Person(); // 静态加载
  ```

* 动态加载

  ```java
  Person p = Class.forName("Person").newInstance(); // 动态加载
  ```
